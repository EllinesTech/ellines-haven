   What: See all site visitors with real IP and location

⚡ QUICK TASKS:

Task: See who's online
  1. Admin → 🟢 Online
  2. See all active users with green dot
  3. Done!

Task: Kick a user offline
  1. Admin → 🟢 Online
  2. Find user
  3. Click "Force Offline"
  4. Done! (they'll log out on next action)

Task: Edit an order
  1. Admin → 🧾 All Orders
  2. Find the order
  3. Click "Edit"
  4. Change anything
  5. Click "Save"
  6. Done! ✅

Task: Unlock a book for customer
  1. Admin → 🧾 All Orders
  2. Find order
  3. Click "Edit"
  4. Change Status to "Completed"
  5. Click "Save"
  6. Done! 📚 Book is unlocked

Task: Check customer IP
  1. Admin → 🟢 Online
  2. Search for customer name
  3. See their IP
  4. Click "View on Map" for exact location
  5. Done!

Task: Verify visitor data
  1. Admin → 📊 Visitors
  2. See real IP, city, country, ISP
  3. Not dummy data anymore!
  4. Done!

💡 TIPS:

Tip 1: Use search in Online Users
  • Search "197.232" to find all users from that ISP range
  • Search "Kenya" to see all from Kenya
  • Multiple users from same IP = shared network

Tip 2: Check statistics
  • High "Online Now" = Busy hours
  • Compare with other times to find peak usage

Tip 3: Unlock books quickly
  1. Admin → All Orders
  2. Find order
  3. Edit → Status: Completed → Save
  4. Takes 30 seconds

Tip 4: Monitor suspicious activity
  • Same IP multiple times = might be bot
  • Timezone mismatch = might be VPN
  • Multiple charges same card = fraud check

Tip 5: Export for analysis
  • Export CSV every week
  • Use for accounting
  • Track which promos are working
  • Identify top customers

❓ FAQ:

Q: How often does Online Users update?
A: Every 30 seconds (automatic)

Q: Can I undo an order edit?
A: No, but you can edit it again. (No undo in database)
   Tip: Take a screenshot before editing important orders

Q: Does Force Offline delete their account?
A: No, just logs them out. They can log back in anytime.

Q: Are visitor IPs accurate?
A: 95% accurate at city level. VPN hides real location.

Q: When do books unlock?
A: When you mark order Status as "Completed" → Immediate

Q: Can customers see their IP?
A: No, only admins can see this.

Q: What if user IP is blank?
A: They haven't visited site yet (registered but never logged in)

Q: How do I see deleted users?
A: You can't. Once deleted, they're gone. Be careful!

Q: Can I edit orders for past months?
A: Yes! You can edit any order, anytime.

Q: Does editing order notify customer?
A: No automatic notification. Tell them yourself if needed.
