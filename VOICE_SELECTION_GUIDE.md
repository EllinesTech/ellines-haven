# 🎤 Voice Selection Feature — Complete Guide

## Overview

The Ellines Haven audio reader includes a comprehensive voice selection system that allows users to choose from 30+ different voices, filter by gender, and adjust playback settings to their preference.

---

## How to Use Voice Selection

### Step 1: Open the Reader
1. Navigate to any book in your library
2. Click on the book to open the Reader
3. Scroll down to find the **Audio Player** (purple/blue card)

### Step 2: Access Voice Settings
```
Look for the gear icon (⚙️) in the top-right corner of the audio player
Click it to open the settings panel
```

### Step 3: Choose Your Filter

The voice selection panel shows three filter options:

```
┌─────────────────────────────────────┐
│  Voice filter                       │
├─────────────────────────────────────┤
│  [ ♀ Female ]  [ ♂ Male ]  [ 👥 All ] │
└─────────────────────────────────────┘
```

**Options:**
- **♀ Female** — Shows female voices (15+ options)
- **♂ Male** — Shows male voices (10+ options)
- **👥 All** — Shows all available voices (30+)

### Step 4: Select a Voice

1. Click on the "Voice" dropdown menu
2. A list of voices appears (scrollable if many)
3. Each voice shows:
   - **Voice name** (e.g., "Microsoft Aria", "Google UK English Female")
   - **Language code** (e.g., "en-US", "en-GB")
   - **Quality badge** (✨ = high-quality, 🔵 = Google Neural)
4. Click any voice to select it
5. The audio player immediately uses the new voice

### Step 5: Adjust Additional Settings

```
┌──────────────────────────────────┐
│  Speed × 1.0×                    │
│  [0.75×] [1.0×] [1.25×] ...     │
│                                  │
│  Pitch × 1.0                     │
│  [====●============] (slider)    │
└──────────────────────────────────┘
```

- **Speed Slider** — Choose from 0.5× to 2.0× playback speed
- **Pitch Slider** — Adjust voice tone from 0.5 to 2.0

## Troubleshooting

### "No Voices Found for This Filter"
**Problem:** Voice list is empty when you select Female or Male filter.
**Solution:** Your browser/device might have limited voice support.
- Try the **All** filter to see available voices
- Check if browser has permission to use system voices
- Try a different browser (Chrome has the most voices)

### "Voice Doesn't Sound Right"
**Problem:** Voice sounds robotic or unclear.
**Solution:**
- Try a **neural voice** (marked with ✨ or 🔵)
- Neural voices sound much more natural
- Make sure you're not on a 3G connection (can affect audio quality)

### "Voice Stops During Long Text"
**Problem:** Audio cuts off mid-paragraph.
**Solution:** This is a browser limitation, not a bug.
- Use Chrome or Edge for best reliability
- Try pausing and resuming to continue
- Reduce playback speed (slower voices are more reliable)

### "Voice Selection Not Saving"
**Problem:** You select a voice, but next time it's different.
**Solution:**
- Your browser is clearing local storage
- Check browser privacy settings
- Try disabling "Clear browsing data on exit"
- Use Chrome instead (better storage support)

### "No Female/Male Voices Available"
**Problem:** Only Male or Female filter shows voices, not both.
**Solution:**
- Your device/browser has limited voice selection
- This is normal on some Android or older iOS devices
- Use the "All" filter to see what's available
- Try a different browser (Chrome has most voices)

## Performance Optimization

### How It Works Internally

1. **Voice Discovery** (~50-1000ms on first load)
   - Browser scans available system voices
   - Results are cached for subsequent loads

2. **Voice Filtering** (instant)
   - Female/Male filter runs client-side
   - No server calls, happens instantly

3. **Voice Selection** (instant)
   - Selecting a voice applies immediately
   - Audio plays with new voice right away

4. **Settings Storage** (automatic)
   - Your voice choice is saved locally
   - Persists across browser sessions

### On Mobile
- Voice dropdown uses **fixed positioning** (efficient)
- Touch targets scaled appropriately (**no performance hit**)
- Settings panel stacks vertically (**smooth scrolling**)
