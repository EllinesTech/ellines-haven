STEP 1: Deploy Cloud Functions (MUST DO FIRST)
================================================================================

Run this command:

  firebase deploy --only functions

Expected output:
  ✔  functions[trackVisitor]: http function initialized (trackVisitor)
  ✔  Deploy complete!

What it does:
  - Deploys updated trackVisitor function
  - Now returns full geo data (ip, city, country, isp, lat, lon, timezone)
  - Enables visitor tracking system
  - Enables presence tracking system

If it fails:
  - Check: firebase login
  - Check: You have Node.js installed
  - Check: functions/index.js has no syntax errors
  - Try again in 1 minute

================================================================================
TROUBLESHOOTING
================================================================================

Q: "Online Users panel is empty"
A: Wait 30 seconds (that's the heartbeat interval). Then refresh. If still 
   empty, check Cloud Functions deployed successfully.

Q: "Visitor IP shows empty"
A: Cloud Functions may not be deployed. Run:
   firebase deploy --only functions

Q: "Receipt editing won't save"
A: Check browser console for errors. Make sure order has all required fields.

Q: "Books aren't unlocking when I mark Completed"
A: Check if order has valid book IDs. Check Cloud Function logs for errors.

Q: "Admin panels don't show up"
A: Rebuild frontend:
   npm run build && firebase deploy
